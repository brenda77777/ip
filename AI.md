# AI-Assisted Coding (A-AiAssisted)

This project was developed with assistance from ChatGPT.

## Tools Used
- ChatGPT (OpenAI)

## How AI Was Used

AI assistance was used to:

- Refactor code to improve SLAP compliance
- Improve separation of concerns between Candy, Ui, and ParsedCommand for better OOP design
- Draft Javadoc comments
- Review and improve error handling logic
- Reduce long methods
- Debug


## Ethical Use

All AI-generated suggestions were:
- Reviewed manually
- Modified where necessary
- Tested before integration

AI was used as a learning and productivity tool instead of a replacement for understanding.
